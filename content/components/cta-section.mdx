## Props

- `title`: The main headline of the section.
- `description`: The supporting text below the headline.
- `primaryActionText`: The text for the primary button.
- `primaryActionUrl`: The URL for the primary button.
- `secondaryActionText`: The text for the secondary outline button.
- `secondaryActionUrl`: The URL for the secondary outline button.
