---
title: Breathing Text
description: Variable font typography that breathes autonomously or reacts dynamically to the user's cursor.
category: Text Effects
---

The `BreathingText` component leverages modern CSS Variable Fonts (`font-variation-settings`) to smoothly interpolate font weight and slant. Instead of snapping between fixed font weights (like 400 to 700), it interpolates seamlessly through every micro-weight in between.

Perfect for highly interactive hero sections where typography takes center stage.

## Installation

<CliInstallCommand componentName="breathing-text" />

> [!WARNING]
> This component **requires** the typeface you are using to be a Variable Font. Standard static fonts will ignore `font-variation-settings`. For example, `Inter` supports the `wght` (weight) and `slnt` (slant) axes natively.
