A highly polished, interactive form component toolkit. Built using Framer Motion, it features floating labels, dynamic glassmorphism focus states, and a smooth inline success state transition upon submission. Includes inputs, textareas, checkboxes, and radio buttons.

## Architecture

We export four primary components: `FloatingInput`, `FloatingTextarea`, `FloatingCheckbox`, and `FloatingRadio`. 

These components are un-controlled internally (for seamless visual animations) but can accept standard React `onChange` and `value`/`checked` props for controlled usage inside standard forms.
