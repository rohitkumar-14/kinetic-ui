---
title: Elastic Sidebar
description: A navigation menu that physically stretches and pulls like a rubber band using SVG path morphing.
category: Navigation
---

An advanced off-canvas navigation menu that uses Framer Motion's spring physics to animate SVG cubic bezier curves. When opened, the edge of the sidebar pulls elastically outwards before snapping into a perfectly straight line, providing an incredibly tactile feel.
