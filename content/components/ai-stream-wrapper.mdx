---
title: AI Stream Wrapper
description: A generative AI-style wrapper that makes UI elements appear as if they are being compiled in real-time.
category: Page Transitions
---

The `AIStreamWrapper` component wraps standard React children and reveals them using a staggered, glowing "skeleton dissolve" effect — mimicking the experience of AI generating a user interface in real-time (like Vercel's v0 or Claude Artifacts).

It treats each direct child element as a discrete "block" and reveals them sequentially with:
1. **Skeleton shimmer placeholders** that glow before the content appears.
2. **A scanning line** that sweeps down the page as blocks are compiled.
3. **Edge glow halos** that pulse around each element as it materializes.
4. **A status bar** showing a typing indicator and a live progress counter.

> [!IMPORTANT]
> **Block Granularity:** The wrapper treats each **direct child** as one "block" to reveal. If you want finer-grained control (e.g., reveal each stat card separately), wrap each one in its own `<div>` at the top level of the wrapper's children.
