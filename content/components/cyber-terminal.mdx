---
title: Cyber Terminal
description: An interactive command palette styled entirely like an old CRT monitor terminal.
category: Blocks & Templates
---

The `CyberTerminal` is a heavily stylized command palette built on top of `cmdk`. It features CSS-driven CRT monitor effects (scanlines, screen flicker, phosphor glow) and utilizes the browser's native Web Audio API to synthesize mechanical keystroke sounds with zero latency.

Perfect for developer-focused portfolios, hackathon sites, or any project that leans into the retro-futuristic cyberpunk aesthetic.

> [!NOTE]
> The terminal uses the native `AudioContext` API to generate sound. Browsers require a user interaction (like clicking a button or pressing a key on the page) before audio can be played. The component handles this automatically by attaching an initializer to the capture phase of the click event.
