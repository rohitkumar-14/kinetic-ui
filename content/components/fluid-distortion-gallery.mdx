title: Fluid Distortion Gallery
description: WebGL image gallery with a liquid noise displacement shader transition.

The FluidDistortionGallery is a React Three Fiber component that takes multiple image textures and transitions between them using a custom GLSL noise displacement shader, mimicking liquid physics.
