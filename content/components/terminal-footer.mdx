## Architecture

This component eschews the standard "three-column grid" footer design for something radically different: an interactive split layout.

- **Interaction:** Hovering over any `Link` instantly sets the `activeCommand` state. 
- **Typewriter Effect:** A `useEffect` hook intercepts state changes and incrementally slices the command string via `setInterval`, giving the illusion of a user typing at a blazing fast 40ms per keystroke.
- **Fallbacks:** If a specific `command` is not provided in your links array, it automatically defaults to generating a bash `cd {href}` command for you.
