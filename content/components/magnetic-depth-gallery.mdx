## Architecture

This component calculates **Euclidean distance** directly in a 60fps `useAnimationFrame` loop to achieve its effect:

1. A single `onMouseMove` event listener is attached to the parent container. This writes the cursor's `X` and `Y` coordinates into highly-performant Framer Motion `useMotionValue`s.
2. Every image in the gallery subscribes to these motion values.
3. On every frame, each image calculates its exact center point on the screen and measures its distance to the cursor.
4. Using linear interpolation, the component dynamically scales and sharpens items that are near the cursor, while fading and applying CSS `blur()` to items that are far away.

The result is a stunning "macro lens" depth-of-field effect that reacts instantly to user input.
