---
title: HeroParticles
description: An interactive HTML5 canvas particle background reacting with fluid repulsion forces on cursor proximity.
category: Hero Components
---

The `HeroParticles` component initializes a high-performance rendering loop using HTML5 Canvas to control an array of floating dot nodes and connection threads. When the mouse moves, nodes automatically calculate repulsion vectors relative to the coordinates to create organic shockwaves.

## Performance Notes

Canvas rendering performance is linked directly to Node density. Keeping counts below `120` (and `60` on mobile viewports) keeps drawing times under 1.5ms per tick, which guarantees solid 60fps on average mobile devices.
