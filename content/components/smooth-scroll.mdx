## Overview
A lightweight wrapper around Lenis that provides high-performance, butter-smooth scroll experiences. This is the foundation for creating Awwwards-style premium web experiences.

## Usage

```tsx
import { SmoothScroll } from "@/components/creative/smooth-scroll"
```

```tsx
<SmoothScroll
  duration={1.2}
  smoothTouch={false}
>
  {/* Your app content goes here */}
</SmoothScroll>
```
