## Overview
The `Typewriter` component is a lightweight, dependency-free (other than framer-motion for the blinking cursor) React component that simulates typing text. It can take a single string to type out once, or an array of strings to infinitely loop through, deleting the previous string before typing the next one.

## Usage

### Single String
```tsx
import { Typewriter } from "@/components/creative/typewriter"

export default function SingleString() {
  return (
    <h1>
      Loading system... <Typewriter text="Done." loop={false} />
    </h1>
  )
}
```

### Rotating Array
```tsx
import { Typewriter } from "@/components/creative/typewriter"

export default function Hero() {
  const words = ["Designers.", "Engineers.", "Creators."]
  
  return (
    <h1>
      Built for <Typewriter text={words} cursorClassName="text-blue-500" />
    </h1>
  )
}
```
