## Overview
The most common navigation pattern on Awwwards-winning websites. This navigation bar stays out of the user's way while they read by hiding on scroll down, but instantly reappears the moment they scroll back up, offering easy access to navigation without wasting vertical space.

## Usage

```tsx
import { SmartNavbar } from "@/components/creative/smart-navbar"
```

```tsx
<SmartNavbar threshold={100} glass={true}>
  <div>Logo</div>
  <div>Menu Items</div>
</SmartNavbar>
```
