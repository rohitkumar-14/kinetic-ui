---
title: Floating Navbar
description: A sleek, auto-hiding floating navigation bar with fluid hover indicator physics.
category: Navigation
---

The `FloatingNavbar` is an essential component for modern landing pages. It provides a non-intrusive navigation experience by automatically hiding when scrolling down and revealing when scrolling up. It features a Framer Motion-powered layout animation that seamlessly tracks the user's hover state across items. Now fully upgraded with **3 hover indicator variants**.
