## Architecture

This component is the holy grail of cinematic web design. Instead of dealing with massive arrays of 150+ PNG sequences (which severely bloats bundles), it uses a much more performant method: **Video Scrubbing**.

We leverage `framer-motion`'s `useScroll` to calculate exact page progress, map it to a `0-1` float, and manually set the `HTMLVideoElement.currentTime` to sync perfectly with the user's scroll wheel inside a sticky container.
