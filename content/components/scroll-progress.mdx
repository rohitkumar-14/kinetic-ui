title: Scroll Progress
description: A top-bar or circular floating scroll progress indicator with smooth spring physics, neon glows, and a scroll-to-top interaction hook.
category: Scroll Systems
---

The `ScrollProgress` component indicates the page's current scroll depth using clean, performant SVG drawings and CSS transforms. It supports two main layout configurations (horizontal progress bar or floating circular progress widget) and multiple color schemes with custom sizes, thicknesses, and neon glows.

## Use Cases

### Combined Global Scroll Indicators

You can render both variants together in your main layout file (`app/layout.tsx` or similar) to give the user the best scrolling experience:

```tsx
import { ScrollProgress } from '@/components/creative/scroll-progress';

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        {/* Top-bar progress */}
        <ScrollProgress variant="bar" theme="cyber" height={4} />
        
        <main>{children}</main>

        {/* Circular indicator with scroll-to-top */}
        <ScrollProgress variant="circle" theme="cyber" size={48} hideAtTop={true} />
      </body>
    </html>
  );
}
```
