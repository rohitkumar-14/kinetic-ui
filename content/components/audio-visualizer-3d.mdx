---
title: 3D Audio Visualizer
description: A stunning WebGL audio visualizer that warps a 3D sphere to the frequency of your music.
category: 3D Components
---

The `AudioVisualizer3D` component mounts a React Three Fiber canvas and uses the browser's native `AnalyserNode` (Web Audio API) to generate a real-time reactive 3D mesh. 

It maps the low-end frequencies (bass) to dynamically scale the sphere, and maps the high-end frequencies to distort the mesh using organic noise parameters.

## Installation

<CliInstallCommand componentName="audio-visualizer-3d" />

> [!WARNING]
> WebGL components require a lot of processing power. Do not mount multiple `AudioVisualizer3D` components on the same page at the same time unless absolutely necessary.
>
> Also, if you are passing external audio URLs, the server hosting the audio **must** provide proper CORS headers, otherwise the browser will block the `MediaElementSource` from reading the frequency data!
