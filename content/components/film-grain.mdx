## Overview
Film grain is the hallmark of modern, Awwwards-winning websites. It removes the "sterile" feel of flat digital colors and adds a subtle, tactile warmth.

Instead of loading heavy, repeating PNG or GIF patterns which tank performance and Lighthouse scores, this component generates the noise programmatically using an inline SVG `<feTurbulence>` filter. It sits securely on top of your app with `pointer-events-none` so it doesn't block interactions.

## Usage

For a full-page overlay, place this inside your main `layout.tsx` or `App.tsx`:

```tsx
import { FilmGrain } from "@/components/creative/film-grain"

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>
        <FilmGrain opacity={0.06} blendMode="overlay" />
        {children}
      </body>
    </html>
  )
}
```
