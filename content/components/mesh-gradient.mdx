---
title: Mesh Gradient
description: An animated, fluid mesh gradient background component powered by Framer Motion and heavy CSS blurs.
category: Background Effects
---

The `MeshGradient` component creates a stunning organic gradient background using intersecting, animated SVG blurs. It's an essential element in modern "glassmorphism" or Awwwards-winning hero sections.
