A highly performant, customizable 3D globe component built using `cobe`. Perfect for "Global Reach" sections or hero headers. It features auto-rotation with momentum-based dragging, rich marker support, and multiple visual presets.

## Performance Note

> [!TIP]
> The Globe component uses WebGL. To prevent it from bloating your initial page load bundle in Next.js, always import it dynamically with `ssr: false` as shown in the usage examples above.

<VariantContent match="Cyber Neon">
  ## Cyber Neon Details
  > [!NOTE]
  > The **Cyber Neon** variant achieves its intense glow by increasing the `diffuse` value to `2` and maxing out the `mapBrightness`. We recommend pairing this on a completely pitch-black background for the best contrast.
</VariantContent>

<VariantContent match={["Aurora Green", "Neon Rose"]}>
  ## Colorful Variants
  > [!TIP]
  > When using highly saturated colors like **Aurora Green** or **Neon Rose**, adjusting the `glowColor` property to be slightly darker than the base `color` helps maintain the 3D spherical illusion.
</VariantContent>
