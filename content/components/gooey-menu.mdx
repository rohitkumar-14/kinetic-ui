---
title: Gooey Menu
description: A fluid, interactive menu button that expands out sub-items using SVG gooey filters and physics.
category: Navigation
---

The `GooeyMenu` uses SVG filters (`feGaussianBlur` and `feColorMatrix`) combined with Framer Motion spring physics to create an organic, fluid extraction effect. When toggled, child items "pop" out of the main button. It features a highly customizable layout engine capable of positioning items in radials, arcs, rows, columns, or grids.

## Usage Examples

### Classic Radial
Spreads items evenly in a 360° circle.
```tsx
import { GooeyMenu, GooeyMenuItem } from "@/components/creative/gooey-menu";
import { Plus, Home, Settings, User, Mail } from "lucide-react";

export function RadialMenu() {
  return (
    <GooeyMenu layout="radial" radius={80} color="#818cf8" icon={<Plus />}>
      <GooeyMenuItem label="Home"><Home /></GooeyMenuItem>
      <GooeyMenuItem label="Profile"><User /></GooeyMenuItem>
      <GooeyMenuItem label="Settings"><Settings /></GooeyMenuItem>
      <GooeyMenuItem label="Messages"><Mail /></GooeyMenuItem>
    </GooeyMenu>
  );
}
```

### Bottom Corner Arc
Fans items out in an arc (perfect for fixed bottom-right corners).
```tsx
export function ArcMenu() {
  return (
    <GooeyMenu 
      layout="arc" 
      radius={100} 
      startAngle={180} // Start pointing left
      spreadAngle={90} // Spread 90 degrees up to top
      color="#22d3ee" 
      icon={<Plus />}
    >
      {/* ... items ... */}
    </GooeyMenu>
  );
}
```
