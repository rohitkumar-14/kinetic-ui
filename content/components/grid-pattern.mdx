## Overview
A subtle grid background is practically a requirement for modern developer tools and technical SaaS platforms. It provides architectural scale and structure to otherwise empty sections.

Instead of loading tiled image assets, the `GridPattern` component uses lightweight CSS radial and linear gradients. It also includes an optional radial `mask-image` (the `fadeEdge` prop) which smoothly fades the grid out towards the edges, ensuring that foreground text remains highly legible.

## Usage

```tsx
import { GridPattern } from "@/components/creative/grid-pattern"

export default function TechHero() {
  return (
    <section className="relative min-h-screen flex items-center justify-center bg-black">
      {/* Background */}
      <GridPattern type="lines" size={32} fadeEdge={true} />
      
      {/* Foreground */}
      <div className="relative z-10 text-center">
        <h1 className="text-5xl font-bold text-white">Deploy to the Edge</h1>
      </div>
    </section>
  )
}
```
