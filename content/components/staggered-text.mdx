## Overview
The `StaggeredText` component is an essential utility for modern scrollytelling sites. Instead of manually splitting text and creating complex staggered arrays in `framer-motion`, this component automatically handles the string splitting (preserving whitespace wrapping) and orchestrates a beautiful cascading reveal when the text enters the viewport.

## Usage

```tsx
import { StaggeredText } from "@/components/creative/staggered-text"

export default function Section() {
  return (
    <section className="py-20">
      <StaggeredText 
        text="The ultimate solution for digital creators."
        type="word"
        animation="slide-up"
        staggerDelay={0.1}
        className="text-6xl font-bold"
      />
    </section>
  )
}
```
