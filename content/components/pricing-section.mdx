The `PricingSection` arranges premium package cards inside dark-glass surfaces. It has been completely rebuilt to support modular configurations, customizable accent colors, and multiple interactive variants like dynamic mouse spotlights and focus blurring.

## Usage Examples

### Spotlight Hover Variant
A dynamic variant where a spotlight follows your cursor across the cards.
```tsx
import { PricingCards } from '@/components/creative/pricing-cards';

export default function Example() {
  return (
    <PricingCards 
      variant="spotlight"
      color="#10b981" // Custom emerald accent
    />
  );
}
```

### Focus Mode Variant
When you hover over a specific card, the others will dim out and blur to emphasize the focused package.
```tsx
import { PricingCards } from '@/components/creative/pricing-cards';

export default function Example() {
  return (
    <PricingCards 
      variant="focus"
    />
  );
}
```
