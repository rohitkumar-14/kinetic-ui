## Overview
A utility component that triggers smooth entry animations when its children enter the viewport. Supports multiple animation presets (fade, slide, scale, blur) and child staggering.

## Usage

```tsx
import { ScrollReveal } from "@/components/creative/scroll-reveal"
```

```tsx
<ScrollReveal animation="slide-up" duration={0.8}>
  <h1>Reveal me when scrolled into view!</h1>
</ScrollReveal>
```
