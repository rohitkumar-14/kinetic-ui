## Architecture

This component is incredibly lightweight compared to traditional physics engines (like `matter.js`). Instead of simulating 2D bodies, it uses a brilliant illusion:

1. It renders your `children` exactly as normal.
2. When `isDestroyed` becomes `true`, it immediately sets the original component to `opacity-0` (so it maintains its exact layout footprint in the DOM without shifting neighboring elements).
3. Simultaneously, it mounts 5 absolute clones of your component perfectly overlaid on top.
4. Each clone is masked via a CSS `clip-path` polygon to represent a shard of glass.
5. Framer Motion applies gravity, translation, and rotation to each clone individually, causing them to shatter outwards and fall.

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `children` | React.ReactNode | | The component to wrap and shatter. |
| `isDestroyed` | boolean | | State controlling whether the component is whole or shattered. |
| `force` | number | 1 | Multiplier for the explosion radius and gravity speed. |
