---
title: Horizontal Image Accordion
description: A sleek layout where hovering over an image strip smoothly expands it to fill the screen while shrinking the others.
category: Layout & Data
---

A highly fluid discovery experience. Utilizing hardware-accelerated CSS flex transitions layered with Framer Motion text animations, this component presents multiple narrow image strips side-by-side. When the user hovers over a strip, it seamlessly expands into a large focal point while the others organically shrink out of the way.
