## Architecture

This component utilizes Framer Motion's `useScroll` hook combined with the `<motion.path>` component.

By mapping the `scrollYProgress` (a value between `0` and `1`) directly to the SVG's `pathLength` style property, we achieve a flawless, hardware-accelerated "drawing" animation that scrubs back and forth perfectly with the user's scroll wheel. No complex intersection observers required!
