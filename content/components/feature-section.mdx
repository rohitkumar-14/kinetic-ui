The `FeatureSection` component displays product or platform features using an elegant, staggered entrance animation. It now supports multiple structural variants out of the box, including classic grids, bento boxes, and minimal lists.

## Usage Examples

### Bento Box Variant
Uses `md:col-span-2` and `md:row-span-2` Tailwind classes injected into the feature data to create an asymmetric, modern bento grid.
```tsx
import { FeatureSection } from '@/components/creative/feature-section';
import { Zap, Shield, Layers } from 'lucide-react';

export default function Example() {
  const bentoFeatures = [
    { 
      icon: <Zap className="w-5 h-5" />, 
      title: "Lightning Fast", 
      description: "Optimized for speed.",
      className: "md:col-span-2 md:row-span-1" 
    },
    { 
      icon: <Shield className="w-5 h-5" />, 
      title: "Secure", 
      description: "Best-in-class security.",
      className: "md:col-span-1 md:row-span-2"
    },
    { 
      icon: <Layers className="w-5 h-5" />, 
      title: "Modular", 
      description: "Use only what you need.",
      className: "md:col-span-1 md:row-span-1"
    }
  ];

  return (
    <FeatureSection 
      variant="bento"
      features={bentoFeatures}
      color="#10b981"
    />
  );
}
```

### Minimal List Variant
A simplified, borderless layout perfect for embedding within tight spaces or sidebars.
```tsx
import { FeatureSection } from '@/components/creative/feature-section';

export default function Example() {
  return (
    <FeatureSection 
      variant="minimal"
      color="#ec4899"
    />
  );
}
```
