The `SpotlightCursor` component tracks the user's mouse position and renders a dynamic background layer that follows the cursor, illuminating or interacting with the content beneath it. Now with **5 visual effects** and optimized React event handlers.

## Effects

| Effect | Description | Best For |
|---|---|---|
| `glow` | Soft radial gradient flashlight | Subtle illumination |
| `reveal` | Darkens container, reveals underneath | "Scratch off" interactions |
| `border` | Combines soft glow with hard CSS mask | Complex layered UIs |
| `gradient`| Multi-stop complex gradient | Vibrant, colorful themes |
| `halo` | Hollow ring effect | Minimalist dark modes |

## How It Works

The component uses an absolute positioned `<motion.div>` as the spotlight layer sitting between the background and your `children`. On `mousemove`, the local coordinates are captured and used to construct a CSS `radial-gradient` (or `mask-image`). `mix-blend-mode` is utilized in certain effects (like `reveal`) to naturally composite the lighting over underlying grid patterns or text.
