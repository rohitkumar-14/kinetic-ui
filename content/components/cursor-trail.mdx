## Overview
The `CursorTrail` (or Blob Follower) is a heavily utilized effect in creative web design. It acts as an organic, ambient background aura that trails behind the user's cursor. Using `framer-motion`'s spring physics, it lags slightly behind the actual cursor position, creating a sense of weight and elasticity.

## Usage

```tsx
import { CursorTrail } from "@/components/creative/cursor-trail"

export default function Layout({ children }) {
  return (
    <div className="relative min-h-screen bg-black overflow-hidden">
      <CursorTrail color="#ec4899" size={200} blur={80} />
      
      <main className="relative z-10">
        {children}
      </main>
    </div>
  )
}
```
