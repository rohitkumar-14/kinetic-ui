## Overview
While the `Fluid Background` is great for full-canvas simulations, sometimes you just need distinct, organic gradient masses drifting behind your content. The `BlobBackground` component is a staple of modern SaaS design.

It uses Framer Motion keyframes to slowly orbit and morph large, heavily blurred `div`s. It automatically sets `pointer-events: none` and `mix-blend-screen` to ensure it sits elegantly behind your UI.

## Usage

```tsx
import { BlobBackground } from "@/components/creative/blob-background"

export default function Hero() {
  return (
    <section className="relative min-h-screen flex items-center justify-center bg-black overflow-hidden">
      <BlobBackground 
        colors={["#ffaa40", "#9c40ff", "#00c6ff"]} 
        blur={80} 
      />
      
      <div className="relative z-10 text-center">
        <h1 className="text-6xl font-bold text-white">Next Generation Tools</h1>
      </div>
    </section>
  )
}
```
