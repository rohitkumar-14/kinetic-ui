## Architecture

Standard chat interfaces pass `strings` or `markdown`. With modern LLMs utilizing "Tool Calling" or structured JSON outputs (like Vercel AI SDK), we can instead pass fully interactive React Components (often called *Artifacts*) directly into the chat flow.

This container allows the `content` of a message to be a `React.ReactNode`. It handles auto-scrolling, spring animations for new bubbles, disabled states, and glowing inputs out of the box.
