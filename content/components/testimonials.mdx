## Overview

The Testimonials block provides an infinitely scrolling wall of customer reviews. It's built with CSS animations for perfect performance and features a premium frosted glass design that automatically adapts to your site's color scheme.

## Usage

```tsx
import { Testimonials } from "@/components/creative/testimonials";

export default function Page() {
  return (
    <Testimonials 
      color="#4f46e5" 
      speed={1.5} 
      scale={0.95} 
    />
  );
}
```
