## Architecture

This component wraps the excellent `cmdk` library with Framer Motion animations and a premium, glowing aesthetic tailored for AI integrations. 

When a user types a query, it dynamically injects an "Ask AI" command at the top of the list. Selecting this command triggers an internal generative state, which seamlessly animates the command list closed and expands a loading indicator, followed by the generated response.

> **Note:** To make this a global shortcut (like `Cmd+K`), simply wrap the `<Command>` element inside a `<Command.Dialog>` as specified by the `cmdk` documentation. We've built this as an inline component so it renders safely inside this documentation page without hijacking your keyboard!
