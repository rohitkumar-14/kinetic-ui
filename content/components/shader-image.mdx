---
title: Shader Image Reveal
description: High-performance WebGL image component with GLSL shader hover transitions.
category: 3D Components
---

The `ShaderImage` component replaces standard `<img />` tags with an interactive 3D WebGL Canvas powered by React Three Fiber. It allows you to seamlessly transition between two images on hover using custom GLSL fragment shaders like **Liquid Ripple** or **Digital Noise Dissolve**.

## Installation

<CliInstallCommand componentName="shader-image" />

> [!WARNING]
> This component relies on `@react-three/fiber` and `three`. Because the images are painted onto a WebGL Canvas via WebGL textures, any external image URLs **MUST** be served with proper CORS (Cross-Origin Resource Sharing) headers, otherwise the canvas will throw a security exception.
