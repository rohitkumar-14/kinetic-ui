---
title: Shiny Text
description: A lightweight, CSS gradient animation that sweeps across text to create a metallic or glowing shine effect.
category: Text Effects
---

The `ShinyText` component uses Framer Motion and `background-clip: text` to animate a linear gradient across text. It's perfect for highlighting premium features, drawing attention to calls-to-action, or adding subtle polish to headings. Now updated with **4 built-in animation variants**.
