---
title: Magnifying Glass
description: An x-ray lens that reveals a secret layout underneath your cursor.
category: Mouse Interactions
---

The `MagnifyingGlass` component creates a stunning "x-ray" or "flashlight" effect by rendering two completely different layouts on top of each other. 

It uses `framer-motion` to track the user's cursor with spring physics, and applies a dynamic CSS `mask-image` with a radial gradient to the top layer. The result is a smooth, performant lens that reveals the secret `lensContent` exactly where the user hovers!

> [!TIP]
> **Performance:** We use native CSS custom properties (`--lens-x`, `--lens-y`) inside a Framer Motion `onUpdate` callback to update the mask position. This bypasses the React render cycle completely, guaranteeing 60fps tracking even if the underlying layouts are extremely complex.
