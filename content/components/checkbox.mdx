---
title: Checkbox
description: A control that allows the user to toggle between checked and not checked.
category: Forms
---

The `Checkbox` component is used in forms when a user needs to select one or more items from a list, or to indicate a boolean value (e.g., "I agree to the terms"). It is built on Radix UI's Checkbox primitive.
