## How it works

This component builds a robust "Operating System" layer inside the browser using React Context and Framer Motion.

1. **`DesktopContext` Z-Index Management**: The `DesktopEnvironment` acts as a context provider tracking the `highestZ` variable. Whenever a user clicks on *any* `DraggableWindow`, that window calls `bringToFront()`, which increments `highestZ` by 1 and assigns it to itself, instantly popping it above all other windows.
2. **`useDragControls`**: By attaching Framer Motion's drag controls specifically to the top title bar of the window, we prevent the user from accidentally dragging the window when they try to highlight text inside the window body.
3. **`dragConstraints`**: The `DesktopEnvironment` passes its `ref` down through the context, so windows are strictly constrained to the desktop boundaries and cannot be dragged off-screen.
4. **Window State**: Supports minimize (hides the component), maximize (animates width/height to 100%), and close (calls the `onClose` prop to unmount it).
