## Architecture

This component is a native WebGL primitive built with **Three.js** and **React Three Fiber**. It maps a 2D image texture onto a plane geometry and uses a custom GLSL fragment shader to calculate the Euclidean distance from the user's cursor (`uMouse`).

As the cursor moves, it generates a sine-wave ripple effect dampened by the exponential distance from the pointer, creating a viscous fluid effect that executes on the GPU for maximum performance.
