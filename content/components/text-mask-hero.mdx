---
title: Text Mask Hero
description: A cinematic hero section featuring a massive bold heading where an abstract video or animated CSS gradient plays inside the text letters.
category: Hero Components
---

The `TextMaskHero` uses advanced CSS techniques (`background-clip: text` and `mix-blend-mode: multiply`) to create stunning visual cutouts. It allows you to pass a video URL or rely on a pure CSS animated gradient to breathe life into your typography.
