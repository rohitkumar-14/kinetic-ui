## Overview
While infinite scrolling marquees are great for logos or sheer volume, high-impact quotes often require focus. The `TestimonialCarousel` puts a single review front and center. 

It features `AnimatePresence` cross-fading, interactive pagination dots, star ratings, and physics-based swipe handling on touch devices (or mouse dragging on desktop).

## Usage

```tsx
import { TestimonialCarousel } from "@/components/creative/testimonial-carousel"

export default function SocialProof() {
  const reviews = [
    {
      id: 1,
      author: "Jane Doe",
      role: "CEO",
      company: "Acme Corp",
      avatar: "/avatars/jane.jpg",
      text: "This UI library changed everything for us.",
      rating: 5
    }
    // ...
  ]

  return (
    <section className="py-20 bg-black">
      <TestimonialCarousel testimonials={reviews} autoPlay interval={6000} />
    </section>
  )
}
```
