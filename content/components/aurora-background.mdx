---
title: Aurora Background
description: A stunning, animated aurora gradient background that creates a subtle, shifting mood.
category: Background Effects
---

The `AuroraBackground` uses an intricate combination of CSS gradients, SVG noise filters, and a repeating linear gradient animation to create a beautiful, organic aurora effect. It is completely driven by CSS and requires zero JavaScript to animate, ensuring maximum performance.
