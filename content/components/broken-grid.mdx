## Overview
A raw, editorial-style asymmetrical grid. This "Tactile Brutalism" approach breaks traditional 12-column layouts by allowing items to span irregular columns and rows, with custom offset transforms to create intentional overlaps.

## Usage

```tsx
import { BrokenGrid, BrokenGridItem } from "@/components/creative/broken-grid"
```

```tsx
<BrokenGrid columns={12} gap={4}>
  <BrokenGridItem colSpan={6} rowSpan={2} zIndex={10}>
    <img src="/hero.jpg" alt="" />
  </BrokenGridItem>
  
  <BrokenGridItem 
    colStart={5} 
    colSpan={4} 
    offsetX="-20px" 
    offsetY="40px" 
    zIndex={20}
  >
    <div className="bg-white p-6 shadow-xl">
      Overlapping Text Box
    </div>
  </BrokenGridItem>
</BrokenGrid>
```
