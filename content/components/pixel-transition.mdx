---
title: Pixel Transition
description: A stunning WebGL shader effect that transitions between two images using a pixelated distortion grid.
category: 3D Components
---

The `PixelTransition` component uses React Three Fiber and custom GLSL shaders to create a mesmerizing hover effect. When hovered, the first image breaks apart into a grid of pixels to reveal the second image underneath, a signature effect seen on top-tier Awwwards portfolios.
