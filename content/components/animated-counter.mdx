## Overview
A high-performance number counter that animates from 0 to a target value using Framer Motion springs. It automatically triggers when scrolled into view. A highly requested component for landing pages, dashboards, and portfolio statistics sections.

## Usage

```tsx
import { AnimatedCounter, StatsSection } from "@/components/creative/animated-counter"
```

### Basic Counter

```tsx
<AnimatedCounter value={500} prefix="+" duration={2} />
```

### Full Stats Section

```tsx
<StatsSection
  stats={[
    { value: 500, label: "Projects Completed", prefix: "+" },
    { value: 99, label: "Client Satisfaction", suffix: "%" },
    { value: 2.5, label: "Revenue Generated", prefix: "$", suffix: "M", decimals: 1 },
    { value: 15, label: "Years Experience" }
  ]}
/>
```
