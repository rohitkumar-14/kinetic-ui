The `TextScramble` component animates text into view by rapidly cycling through a set of randomized characters. It creates a classic "cyberpunk" or "decoding" effect. Now updated with **4 pre-configured character sets** and better re-triggering logic.

## Custom Triggers

You can use the `trigger` prop in combination with state to re-run the animation when a user interacts with an element.

```tsx
import { useState } from 'react';
import { TextScramble } from '@/components/creative/text-scramble';

export function HoverScramble() {
  const [isHovered, setIsHovered] = useState(false);

  return (
    <button 
      onMouseEnter={() => setIsHovered(true)}
      onMouseLeave={() => setIsHovered(false)}
    >
      <TextScramble 
        text="Hover Me" 
        trigger={isHovered} 
      />
    </button>
  );
}
```
