The `BentoGrid` is an incredibly popular layout pattern inspired by bento boxes. Our implementation provides a `BentoGrid` container that handles staggered entrance animations, and a `BentoCard` component that provides a beautiful glassmorphism aesthetic with **3 interactive hover variants**.

## Architecture Note

The grid is composed of two components:
1. `<BentoGrid />`: The outer container. It sets up the CSS Grid (defaults to 3 columns on desktop) and orchestrates the entrance animation.
2. `<BentoCard />`: The individual cells. You can easily make a card span multiple columns by passing Tailwind classes like \`className="@md:col-span-2"\`.
