## Overview
The `BeforeAfterSlider` is an essential component for portfolios in architecture, interior design, photography retouching, and product design. It uses Framer Motion's high-performance `useMotionValue` to bind pointer drag events directly to a CSS `clip-path` mask, ensuring silky smooth, 60fps dragging mechanics on both desktop and touch devices.

## Usage

```tsx
import { BeforeAfterSlider } from "@/components/creative/before-after-slider"

export default function CaseStudy() {
  return (
    <section className="py-20 max-w-5xl mx-auto">
      <BeforeAfterSlider 
        beforeImage="/projects/wireframe.jpg"
        afterImage="/projects/final-design.jpg"
        beforeLabel="Wireframe"
        afterLabel="Final Product"
        className="w-full aspect-video rounded-2xl shadow-2xl"
      />
    </section>
  )
}
```
