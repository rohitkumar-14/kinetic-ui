## Overview
A responsive, performant masonry layout using CSS multi-columns. This ensures items of varying heights fit perfectly without awkward gaps, avoiding the heavy JavaScript calculations of traditional masonry libraries.

## Usage

```tsx
import { MasonryGrid } from "@/components/creative/masonry-grid"
```

```tsx
<MasonryGrid
  columnWidth={250}
  gap={4}
  animate={true}
>
  <img src="/tall.jpg" alt="" />
  <img src="/short.jpg" alt="" />
  <img src="/medium.jpg" alt="" />
  {/* More items */}
</MasonryGrid>
```
